## 🎨 Prinsip Desain UI/UX yang Diterapkan

### 1. Hierarki Visual
- Penggunaan ukuran font yang bervariasi (H1 → H2 → H3 → body) untuk memandu perhatian pengguna
- Warna gradient ungu–emerald sebagai aksen utama yang konsisten di seluruh halaman

### 2. Konsistensi Desain
- Design tokens (CSS Custom Properties) digunakan secara konsisten untuk warna, spacing, radius, dan transisi
- Komponen UI (card, button, badge, section header) memiliki pola visual yang seragam

### 3. Feedback & Affordance
- Hover effects pada semua elemen interaktif (card, button, link)
- Micro-animations untuk merespons interaksi pengguna secara real-time
- Form validation dengan visual feedback (border merah + shake animation)
- Loading screen dengan progress bar sebelum konten dimuat

### 4. Navigasi yang Jelas
- Navbar sticky dengan highlight link aktif berdasarkan posisi scroll
- Hamburger menu responsif untuk layar kecil (mobile/tablet)
- Tombol "Back to Top" di footer
- Smooth scroll ke semua section

### 5. Keterbacaan & Tipografi
- Font `Outfit` (display/heading) dan `Inter` (body) dari Google Fonts
- Kontras warna teks yang optimal (WCAG-friendly)
- Line-height dan letter-spacing yang nyaman dibaca

### 6. Responsivitas
- Breakpoints: 1100px, 900px, 640px, 400px
- Layout grid yang adaptif (dari 3 kolom → 2 kolom → 1 kolom)
- Hamburger menu pada tampilan mobile

## 🚀 Cara Menjalankan

### Lokal
1. Clone repositori:
   ```bash
   git clone https://github.com/adamlennoakmalshahluan-design/portfolio.git
   ```
2. Buka `index.html` di browser apa pun — tidak perlu server!

### Deploy (Vercel / Netlify)
1. Push ke GitHub
2. Hubungkan repo ke [Vercel](https://vercel.com) atau [Netlify](https://netlify.com)
3. Deploy otomatis setiap push
